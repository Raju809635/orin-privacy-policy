# ORIN Privacy Policy Site

Standalone static site for Play Store Privacy Policy URL.

## Deploy on Vercel (separate project)
1. Create a new GitHub repo (example: `orin-privacy-policy`).
2. Push this folder to that repo.
3. In Vercel: `Add New -> Project` -> import that repo.
4. Deploy (no build settings needed).

## URL to use in Play Console
`https://<your-privacy-domain>/privacy-policy`

## Local preview
Open `privacy-policy.html` directly in browser or use any static server.
